# HNG Task 3 Project

This project creates a simple image gallery applicaation with authentication using clerk. It makes use of react dnd kit for the drag and drop functionality

## Setup Instructions

The project requires VSCode or any other code editor installed.

To run the project locally:

Clone the repository

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev

### Install Dependencies

npm install

### Run Dev Server

npm run dev

Then use the username user@example.com with the password 1Password to sign in
```
